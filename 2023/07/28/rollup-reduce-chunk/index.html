此时，打包出来的结果就又变成四个 chunk 了。原因在于模块级别的函数调用会被编译器看做是副作用，会影响 chunk 的合并。比如，假设现在 d 依赖 c，引入 c 时会打印 execute。如果把 b 和 c 合并以后，则 d 引入 c 时会把 b 中的 console.log('execute') 也执行，这种情况有时候会导致 bug。
