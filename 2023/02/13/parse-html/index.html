阻塞还是不阻塞，做个实验就好了

Posted on 2023-02-13 | In html/css

前言

浏览器在解析 HTML 时，遇到 CSS 和 JS 可能会阻塞住，这个结论网上有很多，可自行搜索。

对于 JS，有如下结论：

  1. JS 加载会阻塞 HTML 的解析和渲染
  2. JS 执行会阻塞 HTML 的解析和渲染

对于 CSS，有如下结论：

  1. CSS 加载不会阻塞 HTML 的解析，会阻塞 HTML 的渲染
  2. CSS 加载不会阻塞 JS 的加载，会阻塞 JS 的执行

下面，我们做几个实验来验证上述的结论。
